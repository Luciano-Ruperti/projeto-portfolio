Minhas Ferramentas

HTML

HTML (HyperText Markup Language) é uma linguagem de marcação usada para definir a estrutura do conteúdo de uma página web. Consiste em uma série de elementos que utilizamos para delimitar ou agrupar diferentes partes do conteúdo para que ele apareça ou atue de determinada maneira. HTML é o componente base da web, permitindo a construção de websites e a inserção de conteúdos, por meio dos hipertextos.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para definir a aparência de uma página web. CSS permite que você separe o conteúdo de uma página web da sua apresentação, permitindo que você altere a aparência de um site sem ter que alterar o conteúdo subjacente. Com CSS, você pode definir estilos para elementos HTML, como fontes, cores, layout e muito mais. O CSS é quem dá a "cara" do website.

Javascript

JavaScript é uma linguagem de programação que permite criar conteúdo dinâmico em páginas web. Ela é usada para implementar itens complexos em páginas web, como mapas interativos, gráficos 2D/3D animados, e muito mais. JavaScript é uma linguagem leve, interpretada e baseada em objetos com funções de primeira classe. Podemos dizer que JavaScript traz "ação" para o website.
